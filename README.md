# PokéDex App

## Mis see on?

PokéDex sisaldab detailseid andmeid Pocket Monsterite (tuntud kui Pokémonide) kohta. 
Käesantud programm sisaldab kõikide (1025) pokemonide põhiandmeid, tüüpe ja erivõimeid võttes kasutusse vabalt saadaval PokéAPI API v2 (https://pokeapi.co/docs/v2). 

## Kuidas programmi kasutada?

Programm on võimalik avada järgmisest lingist: https://lynriston.github.io/VeebiAlused_APIcallPohinePage/. Lehe avades viib link teid Avalehele(Home) kus on võimalik otsida soovitud pokémoni kasutades otsingu riba või kerides keskmise hiirega soovitud olendile.
Pokémon kaardile vajudates viiid see detail vaatesse kus saab näha pokemoni nime, pilti, tüüpi, erivõimet ja põhiandmeid. Veel saab vabal tahtel salvestada olend lemmikutesse, vajudaes "Add to favorites" nupule.
Õlevalt navigatsiooni ribalt saab valida avalehe ja lemmikute lehe vahelt. Lemmikute lehelt saab kas minna olendi detail vaatesse või eemaldada lemmikutest.
